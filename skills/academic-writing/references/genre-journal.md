# 期刊论文详细模板(中文核心期刊 / 英文 SCI)

适用于水利、水电学科的期刊投稿与会议论文。相比学位论文,期刊论文短小、聚焦单一贡献,
但行文规范(双语要素、图表编号、量化表达、引用)与学位论文一致。

## 完整结构(按顺序)

1. **题目**:中英文。准确、具体、含关键信息(研究对象 + 关键量 + 方法/条件),不冗长。
2. **作者与单位**:含通讯作者、单位英文名。
3. **摘要 / Abstract**:中英文对照,200~300 字。结构 = 目的/背景(1~2 句)→ 方法
   (1~2 句)→ 主要结果(带关键数据,2~4 句)→ 结论/意义(1 句)。摘要要能独立读懂,
   给出**具体量化结果**,不写"取得较好效果"这类空话。
4. **关键词 / Keywords**:各 4~6 个。
5. **引言(Introduction)**:压缩版综述——研究背景与意义 → 国内外研究现状(分主题、
   带评述、密集引用)→ 现有研究不足 → 本文目的与贡献。通常 1 页以内。
6. **材料与方法 / 实验 / 模型(Methods)**:实验装置与工况、材料参数、量测方法;或
   数值模型(控制方程、紊流模型、计算域与边界条件、网格、验证)。详略以"可复现"为准。
7. **结果与分析(Results)**:按科学问题分小节。每个结果:先文字引出图表 → 定性描述
   现象 → 定量分析(数据、区间、趋势)→ 必要的机理解释。图表必须在正文被解读。
8. **讨论(Discussion)**(SCI 常单列,中文期刊有时并入结果):与已有文献对比、机理
   深入、适用范围与局限。
9. **结论(Conclusions)**:分条(1、2、3)概括核心发现,每条动词短语领起 + 量化结果,
   不重复摘要原句、不引入新数据。
10. **参考文献**:按引用顺序著录,格式按目标期刊要求(投稿前核对该刊模板)。

## 与学位论文的差异
- 篇幅:期刊论文聚焦**一个**核心贡献;学位论文覆盖多个问题、多章结果。
- 综述:期刊用"引言"浓缩,学位论文用独立"绪论"章。
- 无"本章小结""技术路线图""主要符号对照表"等学位论文专属部件(符号在首次出现处定义)。
- 图表数量受限,择要呈现。

## 投稿前必查
- 目标期刊的字数、图表数、参考文献格式、摘要结构要求(不同刊差异大,以官网模板为准)。
- 中英文摘要信息是否等价。
- 每张图表是否都在正文被引用并解读。
- 结论是否分条、量化、且不与摘要简单重复。
- 是否存在未核实或编造的引用(一律标 `【待补:文献】` 交作者)。
