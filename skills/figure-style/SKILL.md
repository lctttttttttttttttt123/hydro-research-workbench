# 水利/水电论文图表规范

## 做什么
规定水利/水电论文里图和表"该长什么样、怎么标注",让全篇插图风格统一、符合期刊与学位论文
惯例。这是规范层;生成图的代码在 hydro-figure-composer 技能。

## 一、编号与题注
- **编号**:图和表分别用"章号.序号"——图 2.17、表 3.1。全篇连续,与正文引用一致。
- **双语题注**:中文题注在上、英文在下,并列。图题在图下方,表题在表上方。
  - 图:`图 5.13 冲刷纵剖面随时间的变化过程`
        `Fig. 5.13 Variation of the scour longitudinal profile with time`
  - 表:`表 2.1 局部冲刷实验工况`
        `Table 2.1 Experimental cases of local scour`
- 英文题注实词首字母大写或仅首词大写,全篇统一一种。

## 二、正文规则:先引后现、每图必解读
- **先引后现**:正文先"如图 3.1 所示……"再出现该图,严禁图先于文字引用。
- **每图必解读**:每张图表都要在正文里被解读——说明它揭示了什么规律、量级、趋势,不做
  只放图不说话的"图墙"。

## 三、字体规范(中文期刊惯例)
- **中文** → 宋体类衬线体(如宋体 / Noto Serif CJK);**英文与数字** → Times New Roman
  (或与其度量兼容的 Liberation Serif)。
- **变量斜体**:所有物理量符号斜体(*x*、*y*、*z*、*d*、*U*、*τ*),用 mathtext 渲染;
  **下标**规范——描述性下标正体(*d*_s 的 s、*U*_c 的 c),数字下标正体(*d*₅₀)。
- **单位正体**,置于斜线后:`... / cm`、`... / (m·s⁻¹)`;单位与变量不混斜体。
- 图内所有文字(标签、刻度、图例、题注英文)统一遵守上述字体约定。

## 四、坐标、单位与变量
- 坐标轴标注"物理量 + 单位",如 `流速 u / (m·s⁻¹)`、`水深 h / m`、`冲刷深度 d_s / cm`。
- 变量斜体、下标规范(*d*₅₀、*U*_c、*τ*_w),与主要符号对照表一致。
- 无量纲量注明(如 `d_s/D`、`x*/D`),坐标可用无量纲形式。
- 同一物理量全篇单位一致。

## 五、常见图型及样式约定
- **冲淤地形图 / 等值线云图**:平面或三维床面高程,发散型配色(冲刷—淤积以零线分色),
  标注坐标与色标(高程/冲深,单位 cm 或 m)。
- **流场矢量图**:速度矢量 + 背景填色(流速大小或平均动能),箭头密度适中不糊。
- **涡量 / 紊动量分布**:发散配色表示正负涡量,或顺序配色表示紊动强度/TKE,附色标。
- **剖面/时程曲线**:如冲刷剖面随时间、冲刷深度随时间——多条曲线用不同线型+图例区分,
  优先线型/标记区分以适应黑白打印。
- **级配曲线**:横轴粒径(对数)、纵轴小于某粒径百分数,标出 d₅₀ 等特征粒径。
- **调度/负荷曲线**(水电系统类):负荷分配、出力过程、耗水流量曲线、库水位过程——
  横轴时间(小时/分钟级),多电站/机组用图例区分,必要时双纵轴。
- **示意图 / 技术路线图**:结构或流程示意,线条简洁、标注清晰,中文为主。

## 六、配色与可读性
- **物理场**(流速、涡量、紊动、冲淤高程)用感知均匀 colormap:强度类用顺序型
  (viridis/cividis),正负量用发散型(RdBu_r),避免彩虹(jet)。
- **多序列对比**(如高桩承台 vs 单桩、方法 A vs B、各电站)用离散语义色序,原则:
  - 主对象(本文提出的结构/方法、重点工况)用冷色(深蓝);对比/基准对象用暖色(红)。
  - 同族保持一致:相关的基准系列归一个冷色族,主对象及其变体归一个族;类别多时先降
    饱和度,再考虑加色。绿/红保留给增益、下降、阈值等"有方向含义"处。
  - 同一对象在不同子图里不要换成另一色族(全篇一致,读起来像"一张图"而非拼凑)。
  - 消融/递进用"单色 + 渐变透明度(alpha)"表达强弱,而非多种色相。
- **黑白可辨**:多曲线优先用线型+标记组合;柱状/面积可加填充线纹(hatch)区分,保证
  灰度打印仍可辨。
- 色块/柱内标注数值时,按背景亮度自动取黑/白文字(见 figure-composer 的 text_contrast)。
- 字号保证印刷后清晰;位图导出 ≥300 dpi(密集多子图柱状图可 600)。

## 七、导出格式(投稿要求)
- **SVG 为首选**:文本保持为可编辑 `<text>` 节点(需 svg.fonttype='none'),可无损缩放、
  便于在 Illustrator/Inkscape 里微调标签对齐。**先存 SVG。**
- **PDF** 为矢量副本(pdf.fonttype=42,文本可编辑)。
- **PNG** 作快速预览/投稿门户,dpi=300(密集柱状图 600)。
- figure-composer 的 `save_pub(fig, 路径)` 一次导出三种格式。

## 八、动态坐标范围
数值集中在窄带(如 80~95)时不要用固定 0~100,应把 y 轴收紧到数据范围(留边距),
以显出差异(见 figure-composer 的 tighten_ylim)。

## 九、一致性检查
全篇插图应满足:编号连续、题注双语且格式统一、坐标单位规范一致、配色风格统一、
每图都在正文被引用并解读。成图前后都对照本规范核对。

## 与其他技能的配合
- 用代码把这些规范落成真实的图 → hydro-figure-composer(内含符合本规范的 matplotlib 样式)。
- 图放进论文并配文字解读 → hydro-academic-writing / hydro-paper-narrative。
