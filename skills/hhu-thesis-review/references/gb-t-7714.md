# 参考文献审查清单（GB/T 7714—2015 顺序编码制）

河海博士论文参考文献用**顺序编码制**：正文按引用先后用上标数字标注，文末按相同序号列出。审查分两层——**著录格式**（每一条写得对不对）和**交叉校验**（正文与列表对不对得上）。

---

## 一、交叉校验（先说清楚它的限度）

**这是"尽力而为"项，不是全量保证。**

- 只有当你能**可靠地从文本中提取出正文引用序号**时，才做"引而未列 / 列而未引"的比对（正文引了 [35] 但列表没有第 35 条；或列表有第 60 条但正文从没引用过）。
- 如果文本是从 PDF 抽取的、序号位置错乱或提取不全（很常见——抽取后序号常被打散到行尾或丢失），**不要硬做全量比对**。要**如实告诉用户**：只能抽查、无法保证查全，并建议作者在 Word 里用"交叉引用 / 题注"功能自查序号连续性与一一对应。
- 能可靠判断的**断号、跳号**（如列表从 [40] 直接跳到 [42]）按【硬伤】标出；不能可靠判断的不要臆断。

正文引用序号断链属于跨部分红线第 2 条，但执行时受限于上面这条"尽力而为"原则。

---

## 二、单条著录格式

### 1. 条目基本结构【硬伤/建议】
标准形态：

```
[序号] 主要责任者. 题名: 其他题名信息[文献类型标识]. 其他责任者. 版本项. 出版地: 出版者, 出版年: 起止页码.
```
期刊最常见的简化形态：
```
[序号] 著者. 题名[J]. 刊名, 年, 卷(期): 起止页码.
```
逐条检查有无**硬缺项**：缺著者、缺题名、缺年份、缺来源（刊名/出版者）→【硬伤】；缺页码、缺卷期→【建议】（部分电子文献可接受，见下）。

### 2. 文献类型标识【硬伤/建议】
题名后应有方括号类型标识，常见：

| 标识 | 含义 |
|------|------|
| [J] | 期刊文章 |
| [M] | 专著（图书） |
| [M]// | 专著中析出的文献（文集中的某篇） |
| [D] | 学位论文 |
| [C] | 会议论文集 |
| [C]// | 会议文集中析出的文献 |
| [R] | 报告 |
| [S] | 标准 |
| [P] | 专利 |
| [N] | 报纸文章 |
| [DB/OL] | 联机数据库 |
| [EB/OL] | 联机电子公告/网页 |
| [J/OL] | 联机期刊 |

缺标识或标错（如把会议析出标成 [J]）→【建议】，普遍缺失→【硬伤】。

### 3. 著者著录【建议】
- **三名以上责任者**：只列前三名（或前一名），其后中文加"等"、西文加"et al."。检查全篇是否**统一**——有的写"等"有的全列，就标出。
- **西文姓名**：姓在前、名缩写在后（如 `Smith J K`），通常**不加缩写点、不加逗号**。出现"de Leeuw, J."这类姓后多逗号、风格与其余条目不一致的，按【建议】统一。
- 中文姓名全名照录。

### 4. 卷期页码一致性【建议/可选】
- 中文期刊**期号前导零**混用（一处 `(01)`、一处 `(1)`）→【可选】统一。
- **起止页码连字符**丢失或粘连（"1356-1367"被写成"13561367"，或缺中间连字符）→【建议】核对补全。注意：这类问题也可能是 PDF 抽取造成的假象，若只拿到抽取文本，提示作者核对原文。
- 跨栏页码 `28-30+23` 形式 GB/T 7714 允许，不报错。

### 5. 标点与中英文混排【可选】
- 中文条目用中文标点（顿号、全角逗号/冒号），西文条目用西文标点。
- 中英文文献按**引用顺序统一编号**，不分语种排序（顺序编码制本就如此），不要因为中英混排就提示重排。

### 6. 特殊条目【建议】
- **网络首发 / 预印本**（只有年份、缺正式卷期页码）：可接受，但与正式见刊条目混排时风格要协调；建议见刊后补全卷期页码。
- **开放获取期刊 / 用 DOI 文章号代页码**（如 `e2021WR029662`）：可接受，但应有文章号或 DOI；纯缺页码又无文章号的，建议补。
- 电子资源应带 [type/OL] 标识及（必要时）访问日期与 URL。

---

## 三、给意见时的合规与示范

- **反例**可摘真实"差句/差条目"，但必须**极短（≤25字）并脱去任何作者、单位、课题标识**，不得整段照搬。
- **正例**一律由你**按 GB/T 7714 规则现场生成**一条规范示范（用虚构的著者/刊名即可），不要摘真实论文里的好条目。

示范（正例，全部虚构）：
- 期刊：`[1] 张三, 李四, 王五, 等. 某交汇区水流结构研究[J]. 水利学报, 2023, 54(3): 312-320.`
- 学位论文：`[2] 赵六. 某基础局部冲刷机理研究[D]. 南京: 河海大学, 2022.`
- 英文期刊：`[3] Smith J K, Brown A, Lee C, et al. Scour around pile groups[J]. Journal of Hydraulic Research, 2021, 59(2): 145-158.`
